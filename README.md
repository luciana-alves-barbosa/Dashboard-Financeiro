# 💰 Dashboard Financeiro

Projeto desenvolvido para controle simples de receitas e despesas.

## 🚀 Funcionalidades
- Adicionar receitas e despesas
- Visualizar saldo total
- Listagem de transações

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript
